By executing create.sh in the installer directory the installer package for Linux can be
created, but you have to make sure that all required binaries are available in the directory:
packages/net.mjproductions.routegen/data

It's especially important that all required Qt libraries, plugins, resources, etc. are provided within the lib and
plugins directory. The list below is the list of files that was required to run routegen for Qt 6.7.2 on Ubuntu 22.04.
For other distributions this list is probably similar, but it's not guaranteed to work.

-Dueo to dependency of the GeoTIFF libraries and binaries, users will have to install these themselves using:

 sudo apt install libgeotiff5
 sudo apt install geotiff-bin
